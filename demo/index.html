Part B — live library demo

This page loads the compiled Part B library (validateRecord and describeForm from dist/) and runs it in your browser. Left: a form rendered entirely from describeForm's returned data structure — type into it and press Validate to run the real validateRecord. Right: every sample record checked live, plus the raw data structure the form was built from.

The library contains no client-specific code. The same functions handle all three sample clients and client-d, a shelter the library has never seen — added here as a definition file only.
